
## 6️⃣ Rebase vs Merge (Interview Favorite)

| Feature                | Rebase           | Merge         |
| ---------------------- | ---------------- | ------------- |
| Commit history         | Linear           | Non-linear    |
| Extra merge commit     | ❌ No             | ✅ Yes         |
| Safe for shared branch | ❌ No             | ✅ Yes         |
| Preferred              | Feature branches | Main branches |
